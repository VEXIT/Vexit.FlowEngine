# Vexit.FlowEngine

Vexit.FlowEngine is an experimental .NET library for orchestrating complex workflows and activities. It provides flow definitions, activity registries, input resolution, and state persistence, though it is not currently in active use.

For more details, see [_docs/README.md](_docs/README.md) file.
